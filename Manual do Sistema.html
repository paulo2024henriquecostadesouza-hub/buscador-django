1. Visão Geral do Sistema

O que é o Buscador de Serviços

O Buscador de Serviços é um sistema web desenvolvido para o CCO (Centro de Controle Operacional) da CPLU que permite visualizar, filtrar e monitorar serviços de limpeza urbana em um mapa interativo de São Paulo. O sistema consolida dados de 5 fontes distintas do sistema Selimp (Flip) e disponibiliza as informações geograficamente.

Com ele, é possível acompanhar em tempo real quais endereços possuem serviços registrados, quais já foram executados, quais estão pendentes, e quais setores estão programados para um determinado dia.

2. Como Acessar o Sistema

Iniciando o Servidor

O sistema roda localmente na máquina 10.1.1.27. Para iniciar, execute o arquivo:

📁 \\10.1.1.140\Cco_Centro_de_Controle_Operacional\CCO\Paulo\ Códigos de Automação\Buscador-Django\Iniciar buscador.bat

Ou, se configurado no Agendador de Tarefas do Windows, o servidor sobe automaticamente quando o usuário faz login.

  • 1
    Duplo clique em "Iniciar buscador.bat" Uma janela CMD se abre mostrando o status do servidor.
  • 2
    Aguardar a mensagem de confirmação A mensagem Buscador de Servicos - ON indica que o servidor está ativo.
  • 3
    Acessar pelo navegador Abra qualquer navegador e acesse o endereço abaixo.

3. Funcionalidades do Mapa

Marcadores e Visualização

O mapa exibe marcadores agrupados por coordenada aproximada (~100m). Cada marcador pode representar múltiplos serviços no mesmo endereço. Os marcadores são divididos visualmente em:

🟢 Feito (verde)

Serviços com data de finalização ou data de execução preenchida, ou cujo status contenha palavras como "finalizado", "executado", "concluído", "realizado".

🟠 Pendente (laranja)

Serviços sem data de finalização ou execução e cujo status não indica conclusão. Necessitam atenção ou acompanhamento.

Ao clicar em um marcador, um popup exibe a lista de serviços naquele ponto, com tipo, status, fonte, data de execução e data de finalização.

Filtros Disponíveis

Filtro Descrição Localização
🔍 Endereço / OS Busca por logradouro ou número de OS. Resultado com Enter ou 800ms após digitar. Barra superior
📅 Mês Filtra serviços por mês/ano de execução ou finalização. Barra superior
📂 Fonte SAC, CNC, BFS, ACIC, OUVIDORIA. Ao selecionar, o filtro de Status atualiza automaticamente para mostrar apenas os status dessa fonte. Barra superior
🏷️ Tipo de serviço Filtra por tipo específico (coleta, poda, limpeza, etc). Barra superior
📋 Status Lista dinâmica: mostra apenas os status da fonte selecionada. Barra superior
✓⚡ Feito / Pendente Filtra mostrando somente serviços feitos, somente pendentes, ou todos. Barra superior
🗺️ Programação Exibe pontos de programação de serviços (setores). Com data selecionada, mostra as rotas do dia. Barra superior (direita)
Filtro inteligente de status: Quando uma fonte é selecionada (ex: SAC), o dropdown de status recarrega automaticamente mostrando apenas os status existentes naquela fonte — sem misturar status de diferentes sistemas.

Programação de Serviços

O checkbox Prog. ativa a camada de programação, exibindo marcadores nos setores agendados. Ao selecionar uma data no campo ao lado, o sistema busca todos os setores programados para aquele dia e desenha as rotas completas em cores distintas no mapa.

Cada setor possui informações de: subprefeitura, tipo de serviço, horário, quantidade de operadores e veículos, e próximas datas agendadas.

4. Painel de Estatísticas e Exportação

Painel de Estatísticas

O botão 📊 Stats abre um painel lateral com estatísticas em tempo real, sempre respeitando os filtros ativos:

📊 Totais

Total de serviços, quantidade feitos e quantidade pendentes.

📂 Por Fonte

Barras de quantidade por fonte (SAC, CNC, BFS, ACIC, OUVIDORIA).

🏷️ Top Tipos

Os 10 tipos de serviço mais frequentes com contagem e barra visual.

🗺️ Por Regional

Cards por subprefeitura mostrando número de setores e pontos de programação cadastrados.

Exportação para Excel

O botão ↓ Excel exporta todos os serviços com os filtros ativos para um arquivo .xlsx. As colunas exportadas são:

Nº OSTipo de ServiçoStatusFonte EndereçoLatitudeLongitudeData ExecuçãoData Abertura
O arquivo é nomeado automaticamente com a data: servicos_AAAAMMDD.xlsx
Dica: Use os filtros antes de exportar para gerar relatórios específicos — por exemplo, apenas os serviços pendentes do SAC no mês de março/2026.

5. Importação de Dados

Acesse a tela de importação pelo botão + Importar no canto superior direito do mapa. Existem três formas de atualizar os dados:

5.1 — Importação Manual de Arquivo

Utilize quando quiser importar um relatório baixado manualmente do Selimp. O sistema detecta automaticamente o tipo (CNC, BFS, ACIC, OUVIDORIA, SAC) pelo nome do arquivo.

  • 1
    Baixar o relatório do Selimp Acesse selimp25.sflip.online, filtre o período desejado e exporte o CSV.
  • 2
    Selecionar o tipo de importação Escolha "Relatório Flip" para serviços, ou "Consultas Programadas" para programação.
  • 3
    Enviar o arquivo Clique em "Importar". O sistema lê o CSV, detecta as colunas automaticamente e insere os registros no banco.
⚠ Atenção: O nome do arquivo deve conter o identificador da fonte: CNC, BFS, ACIC, OUVIDORIA ou SAC. Exemplo: FLIP_CONSULTA_BFS_2026.csv

5.2 — Atualização Automática via Selenium

O botão "▶ Baixar e Importar Agora" abre o Chrome automaticamente, faz login no Selimp e baixa os relatórios de todas as 5 fontes sem intervenção manual.

EtapaO que acontece
LoginChrome abre e faz login em selimp25.sflip.online automaticamente
Download CNCNavega para CNC → Consulta → define datas → exporta CSV
Download OUVIDORIANavega para Ouvidoria → define datas → exporta CSV
Download BFSNavega para BFS → define datas → exporta CSV
Download ACICNavega para ACIC → define datas → exporta CSV
Download SAC 156Navega para SAC → define datas → exporta CSV
ImportaçãoCada CSV é importado direto no banco. Arquivos são removidos após importar.
ConclusãoBeep sonoro indica que o processo foi concluído com sucesso
Configuração do período: O campo "Data de início do período" na tela de importação define a data inicial para o filtro no Selimp. O sistema sempre usa a data atual como data final. Padrão: 01/04/2025

5.3 — Reimportar Pasta Flip

Se os arquivos CSV estiverem salvos na pasta Flip/ do projeto, o botão "Limpar e Reimportar Pasta Flip" apaga todos os dados existentes e reimporta todos os arquivos de uma vez, com tratamento de encoding automático.

⚠ Esta operação apaga todos os serviços do banco antes de reimportar. A programação de setores não é afetada.
